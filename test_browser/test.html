TinyData Test suite

Список тестов (откроется в отдельном окне):

  • TinyData test
  • DotForger test
  • Do stress test on big data

Все тесты написаны для связки mocha+chai и работают как в браузере так и в node.js.

Для запуска тестов в node требуется установить все необходимые модули, выполнив команду npm install в корневом каталоге, далее используйте команду mocha из того же корневого каталога.
